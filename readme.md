# Sassy Starter

A Sass boilerplate organized with [SMACSS](https://smacss.com/), and topped with useful scaffold styles and variables with [BEM](http://getbem.com/introduction/) naming conventions.

## Structure
```
stylesheets/
├── utility/
│   ├── variables.scss
│   ├── grid-settings.scss
│   ├── mixins.scss
│   └── extends.scss
│
├── base/
│   ├── reset.scss
│   ├── fonts.scss
│   ├── animations.scss
│   ├── buttons.scss
│   ├── forms.scss
│   ├── layout.scss
│   ├── lists.scss
│   ├── media.scss
│   ├── tables.scss
│   └── typography.scss
│
├── layout/
│   ├── grid.scss
│   ├── sections.scss
│   ├── header.scss
│   └── footer.scss
│
├── modules/
│   └── global.scss
│  
└── themes/
    └── print.scss
```
